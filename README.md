# Pauls-Digital-Enigma-Machine
My project to emulate a historical Enigma 1 machine from WW2 with Arduino. The board chosen to create Version 1.0 is an Arduino MEGA 2560.

This Repository contains the code for the actual Enigma machine (MEGA 2560) and the code for an optional Morse Encoder for Audible feedback, using an Arduino UNO Rev3.
